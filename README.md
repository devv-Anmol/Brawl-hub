# Brawl Hub 🌟

## 🎯 Purpose
Brawl Hub is a dynamic web application that allows users to explore, search, and organize characters (Brawlers) from the game Brawl Stars. It serves as a comprehensive, interactive database for players to compare brawler stats, rarities, and classes.

## 🔌 API Used
This project integrates the public **BrawlAPI** to fetch real-time game data.
- **API Endpoint:** `https://api.brawlapi.com/v1/brawlers`
- **API Documentation:** https://brawlapi.com/#/

## ✨ Planned Features
- **Dynamic Data Rendering:** Fetches the array of brawlers and dynamically generates UI cards displaying their image, name, and core stats.
- **Search Brawler:** Users can search for specific brawlers by name using array filtering.
- **Filter by Rarity:** Users can filter the brawler list by specific rarities (e.g., Legendary, Mythic) using the `.filter()` method.
- **Sort Brawlers:** Users can organize the displayed brawlers alphabetically or by stats using the `.sort()` method.

## 🛠️ Technologies Involved
- HTML5
- CSS3
- Vanilla JavaScript (ES6+ Array Higher-Order Functions)
- Fetch API
